Install ThrillhouseBot

This creates the GitHub App and gives you the credentials the bot needs to run (App ID, private key, webhook secret, OAuth client ID/secret). Do this first — the bot can't start, let alone serve its own installer, until these values are in your .env.

Create the app

  1. Edit manifest.json and replace every <your-host> with your public hostname (no trailing slash).
  2. Serve this repo root locally: 
    java -m jdk.httpserver -p 8081
  3. Open http://localhost:8081/install.html and click the button below.

Already have the bot running?

Then skip the local server: open http://<your-host>:8080/install.html instead. That page auto-detects the host and builds the manifest for you. It's handy for adding the app to another account or re-registering — not for the first install, since the bot needs to be running already.
